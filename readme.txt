WP Facebook Like and Fan Box

== Description ==

Wp  Like Box widget  for WordPress is a social plugin that enables Facebook Page  owners to attract and gain Likes from their own website.

more info : http://wp-plugins-themes.com/wp-plugin/wp-facebook-fan-box-widget-easy-wp-plugin/ 

= The Like Box enables users to: =

    1.See how many users already like this Page, and which of their friends like it too
    2.Read recent posts from the Page
    3.Like the Page with one click, without needing to visit the Page
    4.No api key or application  id, just url to you facebook fan or ap page.


== Installation ==

    * Download “wpvote-facebook-like-box.zip” to your computer.
    * Login as administrator
    * Click on Plugins –> Add New
    * On “Install Plugins” page, clik on “Upload”
    * Browse and select “wpplugin-facebook-like-box.zip” and click on “Install Now”
    * You can activate it now
    * Finally, add the widget to your sidebar through the ‘Appearance > Widgets’ menu
    * Go to your widget and adjust the settings
